## 📖 模組概述

本模組專為具有 JavaScript 背景的開發者設計。透過對比學習方法，幫助您利用現有的 JavaScript 知識快速掌握 Swift 程式設計。我們採用「從已知到未知」的學習方法，將 JavaScript 概念映射到 Swift 對應概念，特別強調 iOS/macOS 開發、型別安全和協議導向程式設計。

- **實踐導向**: 透過動手編碼範例學習

1. **利用現有知識**: 基於您的 JavaScript 基礎
2. **關注型別安全**: 理解 Swift 的強型別系統和可選型別

4. **實踐驅動**: 透過互動式程式碼範例學習
5. **實際應用**: 將概念應用到 iOS/macOS 開發專案中

## 🔄 語言對比概述

### 語法哲學
- **JavaScript**: 動態型別，靈活且富有表現力
- **Swift**: 強型別，安全且現代，支援型別推斷

### 核心概念映射
- **變數**: `let/const` → `let/var` 支援型別推斷
- **函數**: `function` → `func` 支援型別註解
- **類別**: `class` → `class` 支援 `init` 方法
- **模組**: `import/export` → `import` 語句
- **非同步**: `Promise/async-await` → `async/await` 結構化並發
- **型別**: 動態型別 → 強型別支援可選型別

### 執行模型
- **JavaScript**: 事件驅動，單執行緒支援非同步
- **Swift**: 編譯型，多執行緒支援 ARC 記憶體管理

## 🛠️ 開發環境

### 必備工具
- **IDE**: Xcode（iOS 開發必需）
- **套件管理器**: Swift Package Manager
- **版本控制**: Git 與 Xcode 整合
- **程式碼品質**: SwiftLint, SwiftFormat
- **測試**: XCTest 框架
- **文件**: SwiftDoc

### 快速設定
```bash
# 從 App Store 安裝 Xcode
# 建立新的 Swift 專案
xcode-select --install

# 安裝 SwiftLint（可選）
brew install swiftlint

# 建立新的 Swift 套件
swift package init --type executable
```
