## 快速开始

让我们从一个简单的 Hello World 示例开始，看看 JavaScript 和 Python 的语法差异：

<PythonEditor title="Hello World 对比" compare={true}>
```javascript !! js
// JavaScript
console.log("Hello, World!");

// 变量声明
let name = "LangShift";
const version = "1.0.0";

// 字符串模板
console.log(`Welcome to ${name} v${version}`);
```

```python !! py
# Python
print("Hello, World!")

# 变量声明
name = "LangShift"
version = "1.0.0"

# 字符串格式化
print(f"Welcome to {name} v{version}")
```
</PythonEditor>

## 为什么选择对比学习？

作为 JavaScript 开发者，你已经掌握了很多编程概念。通过对比学习，你可以：

1. **快速理解**：利用已有的 JavaScript 知识理解 Python 概念
2. **避免混淆**：明确两种语言的语法差异
3. **提高效率**：减少学习时间，快速上手 Python
4. **加深理解**：通过对比更好地理解编程语言的本质

## 学习建议

1. **循序渐进**：按照模块顺序学习，不要跳步
2. **动手实践**：每个示例都要在编辑器中运行
3. **对比思考**：思考两种语言的异同点
4. **项目驱动**：通过实战项目巩固所学知识
